# Design assets

This directory contains design references and review notes. These files are not loaded by the applications at runtime.

- `brand/`: concept boards, mockups, and the original logo asset sheet.
- `qa.md`: historical visual QA notes.

Production web brand assets live in `apps/web/public/brand`. Native iOS assets live in `apps/ios/App/Assets.xcassets` and `apps/ios/Shared/Resources`.
