# Nightframe brand assets

These production assets use `nightframe_logo_only.svg` as the canonical mark.

| Asset | Use |
| --- | --- |
| `nightframe_logo_only.svg` | Canonical logo geometry; use this as the source of truth |
| `nightframe_logo_white.svg` | White mark for gold, navy, black, or photographic surfaces |
| `nightframe_logo_black_white.svg` | Black mark on white for light and print-style surfaces |
| `nightframe-mark-gold.svg` | Primary mark on navy, black, or photography |
| `nightframe-mark-navy.svg` | Single-color mark on light backgrounds |
| `nightframe-mark-white.svg` | Single-color mark on dark or gold backgrounds |
| `nightframe-lockup-light.svg` | Black-and-white horizontal logo on light backgrounds |
| `nightframe-lockup-dark.svg` | White horizontal logo on dark backgrounds |
| `nightframe-app-icon.svg` | Gold-square, white-mark master for app/PWA icon exports |

Keep clear space around the mark equal to roughly one quarter of its width. Do
not add a corner radius to iOS artwork; iOS applies its own icon mask. The core
palette is Night Navy `#0F1B2D`, Champagne Gold `#DCA34B`, and white `#FFFFFF`.
